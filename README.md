# 🚀 Career Compass – Student Career Recommendation System

> **Discover career paths that match your interests, personality, and aspirations through an interactive Python-based assessment.**

---

## 📖 Overview

Career Compass is a **Python command-line application** that helps students explore suitable career domains by analyzing their responses to a series of carefully designed questions. Instead of using machine learning, it relies on a **rule-based recommendation engine** that assigns weighted scores based on user preferences and identifies the strongest career match.

The project aims to provide not only career suggestions but also practical guidance to help users prepare for their chosen field.

---

## ✨ Features

* 🧠 Personality-based career assessment
* 🎯 Rule-based career recommendation system
* 📊 Visual career match scores using ASCII progress bars
* 💼 Suggested career roles for each recommended domain
* ⭐ Internship readiness assessment with improvement tips
* 🗺️ Structured learning roadmaps
* ⏳ Estimated learning timelines for key skills
* 💰 Approximate entry-level salary snapshots (India)
* 💡 Personalized career advice
* ✅ Robust input validation and error handling
* 🔄 Option to retake the assessment multiple times

## ⚙️ How It Works

The system asks users five multiple-choice questions related to:

1. Preferred activities
2. Favorite academic subject
3. Interests and strengths
4. Preferred work environment
5. Weekend preferences

Each response contributes weighted points to one or more career domains. After processing all answers, the category with the highest score is recommended.

---

## 📋 What the Program Provides

After completing the assessment, users receive:
